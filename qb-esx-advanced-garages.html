A fully featured, lightweight and easy to use multi-garage system for QBCore. Loosely based off the concept of qb-garages, I originally built this script for a roleplay server that I play on frequently, but it came out really nice I thought I should share it here. Here are some of the core features:

  • 💎 Clean, intuitive and super easy to use UI
  • 🚗 Multiple garage locations & easy transfers between garages
  • ✈️ Car, boat and air garages
  • 🚧 Vehicle impound with self-retrieval, price, timer and reason
  • 🧰 Proper job garages - job vehicles are owned vehicles
  • 💀 Gang garages - owned gang vehicles accessible to all members
  • 🏡 Private garages (for houses or other private properties)
  • ✅ Duplication and spawn pileup protection
  • 💻 Admin commands for managing vehicles
  • ✨ Highly customisable config file and callback functions
  • 🌎 Available in English and Dutch out the box + ability to add more
  • ❤️ Responsive, one-to-one personal support available via Discord and free updates for life!
